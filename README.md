# 喜马拉雅解密工具 (增强稳定版)

这是一个基于 Python 的喜马拉雅 `.xm` 音频格式本地解密工具。通过调用底层 WebAssembly 算法，实现快速、无损的音频解密与格式转换。

## 🌟 鸣谢与版本说明 (Credits)

本项目是对 [sld272/Ximalaya-XM-Decrypt](https://github.com/sld272/Ximalaya-XM-Decrypt) 项目的延续与增强。

特别感谢原作者 **sld272** 逆向并提供的核心 WebAssembly 解密算法（`xm_encryptor.wasm`）以及原始 Python 基础逻辑！

**关于持续更新：**
由于原作者的仓库已于 2025 年 8 月归档（停止维护），而原版代码在遇到 0KB 或损坏文件时存在直接闪退的缺陷。为了方便大家进行大批量转码，我在此仓库基础上进行了深度修复并继续维护。本仓库将保留原作者强大的核心算法，主要致力于提升软件的容错率、稳定性和小白用户的易用性。

## 🎁 附赠小工具：音频智能排序重命名 (rename.py)

很多时候，批量解密出来的音频文件因为集数没有“补零”（例如 `1` 而不是 `0001`），会导致在手机、车机或电脑里**排序完全错乱**。为了彻底解决这个问题，本仓库特别附赠了一个强迫症专属清洗脚本。

### 🛠️ 工具功能
1. 自动无视并扒掉原本乱七八糟的旧序号前缀。
2. 像做手术一样精准剔除原文件名里的集数数字。
3. 统一在最前面加上规范的 `[0001]` 格式补零前缀。

### 🚀 如何使用？
1. 确保你的电脑已安装 Python 环境。
2. 下载本仓库中的 `rename.py` 文件。
3. 双击运行 `rename.py`（或在命令行运行 `python rename.py`）。
4. 在弹出的窗口中选择你的音频文件夹，按下回车即可瞬间完成清洗与重命名！
